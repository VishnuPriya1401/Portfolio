Restaurant App

Role: UI Designer

Tool: Figma

Duration: Feb 2023 - April 2023

Goal of the Project

Our food ordering app will allow users to book table in prior and placed orders using this restaurant app. The orders can also be placed using the calling option provided. Users can also set favorite dishes, easy navigation within the app.

This App is designed with the targeted audience - who use apps to order food, busy workers, who wish to book tables in prior.

What's new in this Project

Easy repeat orders

Just repeat your favorite orders by repeat order method, Easy and time saving. You can also tag the order by giving the menu a name and save for easy quick access.

Color palette

The reason for the colors chosen is that, blue with combination 0f white gives an elegant and delight date look to the app, which matches the restaurant’s date theme. To add slight variation in the color to show difference, the shades of colors are added.

Key Challenges of the App

The Main idea of the App is to create a user friendly food ordering and table booking app for an restaurant. The key challenge is to create a menu tag for users to repeat their fav menu and tag it with a name. This app also has a calling feature for quick ordering for users who are busy and can only spend less time in ordering food.
